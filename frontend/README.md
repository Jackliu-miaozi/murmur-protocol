# Frontend

前端应用，提供 Murmur Protocol 的用户界面。

## 功能

- **议题浏览**：查看当前活跃的 Topic
- **实时聊天**：Live Feed 实时消息展示
- **发言功能**：发送消息，消耗 VP
- **精选区**：查看精选评论
- **资产管理**：DOT 质押、vDOT 管理、VP 查看
- **NFT 展示**：查看历史 Topic 的 NFT 记忆
- **热度可视化**：展示 Topic 热度曲线
- **参与画像**：用户参与行为统计

## 核心页面

- 首页：活跃 Topic 列表
- Topic 详情：实时聊天、精选区
- 资产管理：质押、赎回、VP 查询
- NFT 画廊：历史记忆展示
- 数据看板：热度曲线、参与统计
